Quản lý Modal (Hộp thoại)
Bootstrap 5 không cần dùng tới jQuery. Trong code của bạn, Modal được khởi tạo thông qua Vanilla JS:

new bootstrap.Modal(modalElement): Lệnh này biến một thẻ HTML (như <div id="passwordModal">) thành một Object Modal của Bootstrap để có thể điều khiển qua code JS.

.hide() / .show(): Dùng để đóng/mở Modal bằng code. Ví dụ: Khi ấn nút "Lưu mật khẩu mới" (nếu hợp lệ), hàm bootstrapModal.hide() sẽ được gọi để đóng cửa sổ bật lên.

hidden.bs.modal: Đây là Event Listener mặc định của Bootstrap. Nó sẽ "lắng nghe" và kích hoạt sau khi Modal đã đóng lại hoàn toàn (kết thúc cả hiệu ứng trượt/mờ).

Trong code: Bạn dùng nó để tự động xóa sạch dữ liệu cũ (passwordForm.reset()) và reset lại validate form mỗi khi user đóng Modal.

Validation Form (Kiểm tra dữ liệu)
passwordForm.checkValidity(): Đây là hàm mặc định của trình duyệt HTML5, nhưng được tích hợp chặt chẽ với Bootstrap. Nó kiểm tra xem các input đã điền đủ chưa (dựa trên thuộc tính required ở HTML).

was-validated: Một class CSS của Bootstrap. Khi bạn gắn class này vào thẻ <form> (thông qua classList.add("was-validated")), Bootstrap sẽ tự động kích hoạt các style báo lỗi màu đỏ (invalid) hoặc xanh lá (valid) lên viền của các ô input.

System Toast (Thông báo bật lên)
Hàm showToast tạo ra một giao diện thông báo nhỏ gọn ở góc màn hình.

text-bg-${type}: Một class tiện ích của Bootstrap 5 (ví dụ: text-bg-success, text-bg-danger). Nó tự động thiết lập cả màu nền (background) chữ lẫn màu chữ (text color) sao cho đủ độ tương phản, giúp bạn không cần tự code CSS.

data-bs-dismiss="toast": Một thuộc tính HTML đặc biệt của Bootstrap dùng ở nút dấu X. Nhờ nó, Bootstrap tự hiểu chức năng ẩn thẻ Toast này đi khi được click mà không cần bạn phải viết thêm dòng sự kiện click bằng JS.

Dark Mode (Theme)
data-bs-theme: Kể từ Bootstrap 5.3 trở đi, Bootstrap hỗ trợ Dark Mode qua một Data Attribute. Bạn chỉ cần đổi giá trị html.setAttribute("data-bs-theme", "dark") hoặc "light", toàn bộ màu nền, chữ, viền form, nút bấm có sẵn của Bootstrap trên trang web sẽ tự động đảo màu sang tối/sáng tương ứng. Chức năng số 8 trong code đang kiểm soát trực tiếp tính năng này.

Các Class Tiện Ích (Utility Classes) được gen ra từ JS:
Phần lớn các chuỗi HTML bạn tự dùng JS sinh ra (ví dụ trong renderVault và editPwd) đều tận dụng mạnh mẽ các lớp tiện ích của Bootstrap:

d-flex, flex-column, align-items-center: Dùng hệ thống Flexbox của Bootstrap để căn chỉnh vị trí thẻ không cần tự CSS.

shadow-sm, rounded-3: Tạo đổ bóng nhẹ và bo góc cho các dòng chỉnh sửa (Edit Row) nhìn giống thẻ (card) trơn tru hơn.

form-control: Class kinh điển biến thẻ <input> thành các ô nhập liệu theo chuẩn giao diện phẳng của Bootstrap.

btn, btn-outline-secondary: Biến thẻ button thành các nút nhấn đẹp mắt.
